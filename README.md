# 星座数独

这是一个纯静态、单机运行的 9 宫格数独网站。

## 当前玩法

- 简单版：500 题
- 普通版：500 题
- 大师版：500 题
- 玩家可以自由选择难度和题号
- 通关 10 个大师版后解锁十二星座玩法
- 十二星座各有一个 3x3 矩阵，9 个锁代表 9 局数独
- 每个星座矩阵随机搭配简单版、普通版、大师版
- 解开一个矩阵的 9 个锁后显示对应星座图案

## 本地打开

直接双击 `index.html` 即可游玩。

## 让其他人访问

把这个文件夹部署到 GitHub Pages、Cloudflare Pages、Netlify、Vercel 或任意静态服务器即可。站点不需要数据库和后端，进度保存在玩家自己的浏览器本地。
